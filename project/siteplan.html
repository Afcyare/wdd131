HydrateMe - Website Plan

WDD 131 - Dynamic Web Fundamentals

Site Name

HydrateMe - Smart Water Intake Tracker

I chose this name because it clearly communicates the purpose of helping users maintain proper hydration. The name is simple, memorable, and action-oriented, encouraging users to engage with the tool.

Optional domain availability: hydrateme.app or myhydrateme.com

Site Purpose

The HydrateMe website will:

  • Calculate personalized daily water intake goals based on individual metrics
  • Provide an intuitive interface to track water consumption
  • Display visual progress toward hydration goals
  • Offer educational content about hydration benefits
  • Send helpful reminders to drink water throughout the day

Scenarios

Target audience questions:

  1. "How much water should I drink based on my weight and activity level?"
  2. "What are some easy ways to remember to drink water during my busy workday?"
  3. "How can I track my water intake without downloading another app?"

Color Schema

Water-inspired color palette:

Primary Blue (#2a7fba)

Used for: Headings, primary buttons, important accents

Light Blue (#5fb0e8)

Used for: Secondary elements, backgrounds, hover states

Background (#f5f9ff)

Used for: Page background

Typography

Primary font family:

'Segoe UI', Tahoma, Geneva, Verdana, sans-serif - Used for all body text and most content.

Headings will use the same font family with increased weight and size for hierarchy.
